# Учет авторов и книг в литературном клубе
## О проекте
#### Данная система предназначена для организации работы с литературой книжного клуба.

## Цель:
#### Создать организованную систему работы с литературой, которая облегчит работу с книгами литературного клуба.

## Основные пользователи: 
#### 1. Библиотекарь
#### 2. Участник

## Основные функции системы: 
#### 1. Авторизация пользователей
#### 2. Регистрация нового читателя
#### 3. Управление книгами
#### 4. Оформление брони и выдачи
#### 5. Поиск по каталогу
